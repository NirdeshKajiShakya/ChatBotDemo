## Step 1: Go to Groq Console

1. Open this link in your browser:
   [https://console.groq.com/](https://console.groq.com/)

2. Click **"Sign Up"** or **"Login"**

3. You can sign up with:
   - Google account
   - GitHub account
   - Email

---

## Step 2: Create an API Key

1. Once logged in, go to the **API Keys** section:
   [https://console.groq.com/keys](https://console.groq.com/keys)

2. Click **"Create API Key"**

3. Give it a name (e.g., "TestingProject")

4. Click **"Create"**

5. **Copy the key** - it will start with `gsk_`

---

## Step 3: Add the Key to Your Project

1. Open `local.properties` in your project

2. Replace the placeholder:
   ```properties
   GROQ_API_KEY=gsk_your_actual_key_here
   ```
